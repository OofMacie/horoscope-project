# 🔮 Horoscope Project

A fun and interactive horoscope generator built with JavaScript basics from [Codedex's JavaScript Checkpoint Project](https://www.codedex.io/javascript/checkpoint-project/horoscope).

## 📖 About

This project demonstrates fundamental JavaScript concepts including DOM manipulation, event handling, and conditional logic by creating an entertaining horoscope application. Users can select their zodiac sign and receive a personalized horoscope reading.

## 🌐 Live Preview

Check out the live demo here: [illuminatehoroscope.netlify.app)

## ✨ Features

- Interactive zodiac sign selection
- Dynamic horoscope generation
- Responsive and user-friendly interface
- Built with vanilla JavaScript (no frameworks)

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)

## 🚀 Getting Started

1. Clone the repository
2. Open `index.html` in your web browser
3. Select your zodiac sign to generate a horoscope

## 📚 Learning Outcomes

This project helped practice:
- DOM element selection and manipulation
- Event listeners and handling
- Conditional statements
- Array and object handling
- String manipulation and templating
